RESOURCES AND ARTIFICIAL INTELLIGENCE

Data Sources:

NASA JPL Horizons System
Type: Astronomical Data, Ephemerides data
Usage: Real-time positions and velocities of celestial bodies.
Link: JPL Horizons

NASA Small-Body Database (SBDB)
Type: Orbital data of small bodies (asteroids, comets)
Usage: Data for asteroid and comet orbit tracking and plotting.
Link: NASA SBDB

NASA Near-Earth Object (NEO) API
Type: NEO data
Usage: Real-time data for tracking NEOs and their trajectories.
Link: NASA API

Three.js
Type: JavaScript library for 3D graphics
Usage: Used in the rendering of the 3D view of the solar system.
Link: Three.js

HTML5 & CSS3
Type: Web development technologies
Usage: Used to create Three Dimensional Models required for the Solar System Model.

JavaScript (with the help of React Library)
Type: Programming language
Usage: Main language used for developing interactive features and processing data.

Visual Studio Code
Type: Code editor
Usage: For coding and project development.
Link: Visual Studio Code

GitHub
Type: Version control and collaboration platform
Usage: For version control and sharing code.
Link: GitHub

Integrating AI Functionality

Integrating AI functionality into the Virtual Orrery project will add a new dimension to the functionalities and interaction mechanisms of the tool. It allows the end-users to learn as much as possible about astronomy and space exploration in a deeper way. Applications of AI in these applications are going to drastically alter the experience of the user, when complex astronomical data will be accessible for understanding to a broad audience.
